A pill-shaped call-to-action button in Epilogue. Five variants and three sizes; supports leading/trailing icons and link rendering.

```jsx
<Button variant="primary" size="lg" iconRight="arrow-right">Get your free audit</Button>
<Button variant="outline">Learn about SEO</Button>
<Button variant="secondary" as="a" href="#">Book a call</Button>
```

Variants: `primary` (Pesty Red, glows on hover — the main CTA), `secondary` (deep teal), `outline`, `ghost`, `light` (for use on dark sections). Sizes: `sm` / `md` / `lg`. Use `full` to stretch. Icons reference the `Icon` set by name.
