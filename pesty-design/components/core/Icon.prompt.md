Renders a single icon from Pesty's curated Lucide-based set (24px grid, 2px stroke, inherits `currentColor`).

```jsx
<Icon name="arrow-right" size={20} />
<Icon name="shield-check" size={24} style={{ color: 'var(--brand)' }} />
```

Available names: `arrow-right`, `arrow-up-right`, `check`, `phone`, `search`, `menu`, `x`, `star`, `chevron-right`, `chevron-down`, `map-pin`, `trending-up`, `shield-check`, `target`, `zap`. Pair with Button (`iconLeft`/`iconRight`), Tag, and Stat. `star` renders filled; all others are strokes.
