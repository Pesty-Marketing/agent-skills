Bordered checkmark pill for trust points and feature lists ("Built exclusively for pest control").

```jsx
<Tag>Built exclusively for pest control</Tag>
<Tag tone="onDark">100,000+ leads generated</Tag>
<Tag icon="shield-check">Trusted by PCT Top 100</Tag>
```

`tone`: `brand` (default), `neutral`, `onDark` (for dark hero sections). The leading icon defaults to a red check.
