The base surface container. White with soft shadow + border by default; `deep` for dark teal panels, `subtle` for off-white fills, `outline` for bordered-only.

```jsx
<Card>Default white card</Card>
<Card variant="deep" padding="lg">Dark teal panel for stats & CTAs</Card>
<Card interactive>Lifts on hover — use for clickable service/case-study cards</Card>
```

`padding`: `none` / `sm` / `md` / `lg`. Set `interactive` for hover-lift behavior.
