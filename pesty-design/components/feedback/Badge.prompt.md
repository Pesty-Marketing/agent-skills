Status / metadata pill in Poppins. Six variants for different emphasis.

```jsx
<Badge variant="soft">New</Badge>
<Badge variant="solid">PCT Top 100</Badge>
<Badge variant="success">+30% YoY</Badge>
<Badge variant="dark">Case Study</Badge>
```

Variants: `solid` (red), `soft` (red tint — default), `neutral`, `outline`, `dark`, `success`. Sizes `sm` / `md`.
